# Color Card Review App

This project adds a local web review page for `datasets/color_card_roi`.

Purpose:
- Review horizontal and vertical preview images visually.
- Mark whole images or individual boxes as wrong.
- Export a copyable text summary for manual correction.

Data source:
- `datasets/color_card_roi/manifest.csv`
- `datasets/color_card_roi/preview/**`
- `datasets/color_card_roi/labels/**`

Feedback output:
- Structured lines for direct fixes.
- Human-readable summary for quick copy/paste.
