# 功能 3：SPU名称生成不干胶模板需求说明

## 1. 当前状态

该功能暂缓实现。第一阶段只保留需求文档，不进入开发。

## 2. 功能目标

从用户选择的 Excel 文件中读取指定起始单元格往下的连续数据，将读取到的内容写入 Word 不干胶模板，生成新的 Word 文档。

## 3. 页面需求

用户点击首页“SPU名称生成不干胶模板”后进入该功能页面。

页面包含：

1. Excel 输入设置
   - 选择要转换的 Excel 文件。
   - 设置读取起始单元格。
2. Word 模板设置
   - 选择或显示要写入的不干胶 Word 模板。
3. 输出设置
   - 输入转换后的 Word 文件名称。
   - 选择转换后的 Word 保存地址。

## 4. Excel 读取规则

选择 Excel 文件后：

- 只读取当前页中的信息。
- 从起始单元格开始读取。
- 往下读取所有不间断的行。
- 读取这些连续行中的所有列数据。

## 5. Word 写入规则

读取到的数据写入 Word 不干胶模板：

- 从模板第一行开始写入。
- 写满第一行后自动换行。
- 直到所有读取到的单元格内容写完。
- 如果一页模板写不完，自动复制模板生成第二页继续写入。
- 持续复制页面，直到全部数据写完。

## 6. 暂未确认事项

后续实现前需要确认：

- Excel 支持 `.xlsx` 还是也支持 `.xls`。
- “当前页”是否指 Excel 打开时的活动 Sheet，还是用户在软件中选择 Sheet。
- 起始单元格输入格式，例如 `A1`。
- 空单元格是否表示读取结束。
- 多列数据写入不干胶模板时的拼接方式。
- Word 不干胶模板是内置还是用户选择。
