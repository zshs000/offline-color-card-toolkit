# 功能 2：主图截图及名称更改需求说明

## 1. 当前状态

该功能暂缓实现。第一阶段只保留需求文档，不进入开发。

## 2. 功能目标

根据用户选择的图片，识别图片中左上角主图名称位置，按用户选择的尺寸截取图片，并将截取后的图片按识别名称重命名保存。

## 3. 页面需求

用户点击首页“主图截图及名称更改”后进入该功能页面。

页面包含：

1. 截图尺寸选择
   - `10cm * 10cm`
   - `15cm * 15cm`
2. 输出保存目录选择
   - 选择截图及改名后图片保存的地址。
3. 图片选择
   - 选择要截图及改名的图片。

## 4. 处理规则

点击确认后，软件需要：

1. 读取用户选择的图片。
2. 识别图片左上角贴的主图名称位置。
3. 根据识别到的位置和用户选择的尺寸截图。
4. 将截图结果保存到目标文件夹。
5. 截图后的图片名称使用识别到的左上角数字或文字。

## 5. 图片质量要求

- 截取后的图片必须尽量保持原图信息。
- 不应主动压缩导致图片质量明显下降。

## 6. 暂未确认事项

后续实现前需要确认：

- `10cm * 10cm`、`15cm * 15cm` 对应的 DPI 或像素换算规则。
- 主图名称区域是否固定。
- 是否需要识别结果确认页。
- 重名文件的处理规则。
- 支持的图片格式范围。
