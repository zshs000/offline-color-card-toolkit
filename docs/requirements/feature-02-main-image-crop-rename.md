# 功能 2：主图截图及名称更改需求说明

## 1. 当前状态

该功能已实现。

## 2. 功能目标

根据用户选择的图片，识别图片中左上角主图名称位置，按用户选择的尺寸截取图片，并将截取后的图片按识别名称重命名保存。

## 3. 页面需求

用户点击首页“主图截图及名称更改”后进入该功能页面。

页面包含：

1. 截图尺寸选择
   - `10cm * 10cm`
   - `15cm * 15cm`
2. 输出保存目录选择
   - 选择截图及改名后图片保存的地址。
3. 图片选择
   - 选择要截图及改名的图片。

## 4. 处理规则

点击确认后，软件需要：

1. 读取用户选择的图片。
2. 识别图片左上角贴的主图名称位置。
3. 以识别到的名称块左上角为截图起点，根据用户选择的尺寸截图。
4. 将截图结果保存到目标文件夹。
5. 截图后的图片名称使用识别到的左上角数字或文字。
6. 如果识别名称为空，使用原文件名并提示用户。
7. 如果目标文件名重复，自动保存为 `名称-2.jpg`、`名称-3.jpg`。

## 5. 尺寸换算规则

- 优先读取图片自带 DPI 进行厘米到像素的换算。
- 图片没有 DPI 或 DPI 无效时，默认按 `300 DPI` 换算。
- `10cm * 10cm` 和 `15cm * 15cm` 均输出正方形截图。

## 6. 图片质量要求

- 截取后的图片必须尽量保持原图信息。
- 不应主动压缩导致图片质量明显下降。
- JPEG 输出使用最高质量保存。
- PNG 输出保持 PNG 格式。

## 7. 验收标准

- 用户可以从首页进入“主图截图及名称更改”页面。
- 用户可以选择 `10cm * 10cm` 或 `15cm * 15cm`。
- 用户可以选择输出目录和多张图片。
- 点击确认后，目标目录生成按左上角识别名称命名的截图图片。
- 重名图片不会覆盖已有文件。
