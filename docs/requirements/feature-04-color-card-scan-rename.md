# 功能：色卡扫描图改名需求说明

## 1. 当前状态

该功能已实现。

## 2. 功能目标

根据用户选择的色卡扫描图片，识别图片左上角色卡名称位置中的数字或文字，将整张原图复制到指定文件夹并按识别名称重命名。

## 3. 页面需求

用户点击首页“色卡扫描图改名”后进入该功能页面。

页面包含：

1. 输出保存目录选择
   - 文案为“色卡改名后扫描图片保存的地址”。
2. 图片选择
   - 文案为“选择改名的色卡扫描图”。
   - 支持批量选择 `.jpg/.jpeg/.png` 图片。
3. 确认按钮
   - 点击后开始识别和保存。

## 4. 处理规则

点击确认后，软件需要：

1. 读取用户选择的图片。
2. 使用离线 OCR 识别图片左上角的色卡名称。
3. 提取识别到的数字或文字作为输出文件名。
4. 将原图片复制到目标文件夹，不重新编码、不压缩。
5. 如果识别名称为空，使用原文件名并提示用户。
6. 如果目标文件名重复，自动保存为 `名称-2.jpg`、`名称-3.jpg`。

## 5. 图片质量要求

- 扫描图改名只复制原文件字节，不改变图片内容。
- 不主动压缩，不降低图片尺寸或清晰度。

## 6. 验收标准

- 用户可以从首页进入“色卡扫描图改名”页面。
- 用户可以选择输出目录和多张图片。
- 点击确认后，目标目录生成按左上角识别名称命名的图片。
- 重名图片不会覆盖已有文件。
